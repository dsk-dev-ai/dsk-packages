# Support

## Documentation

The [docs/](docs/) directory contains documentation for development, publishing, release process, and coding standards.

## GitHub Discussions

Use [GitHub Discussions](https://github.com/darshankachare/dsk-packages/discussions) for:

- Asking questions
- Sharing ideas
- General discussion about the project

## Bug Reports

If you have found a bug, open a [Bug Report](https://github.com/darshankachare/dsk-packages/issues/new?labels=bug&template=bug_report.md) issue.

## Feature Requests

If you have an idea for a new feature, open a [Feature Request](https://github.com/darshankachare/dsk-packages/issues/new?labels=enhancement&template=feature_request.md) issue.

## Security Issues

Do not open public issues for security vulnerabilities. See [SECURITY.md](SECURITY.md) for the responsible disclosure process.
